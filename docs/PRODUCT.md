# Product definition

## Goal

Build a WoW-style MMO world editor that supports authoring and exporting large tiled worlds.

## Target workflows (v1)

- Create/open project
- Navigate and stream a large world
- Sculpt terrain (raise/lower/smooth/flatten)
- Paint surface materials (weightmaps)
- Author liquids (lakes/rivers/oceans)
- Place props/doodads (with snap)
- Export runtime artifacts
- Artifact-only preview mode

## Non-goals (v1)

- Real-time collaborative editing
- Full quest/scripting IDE
- In-editor modeling

## Definition of done (v1)

- Stable save formats with migrations
- Streaming-enabled editing at MMO scale
- Deterministic export pipeline
- Reliable autosave and crash recovery
- Usable UX for content creators
