# Editor Controls

This list reflects the current bindings. Most hotkeys require the viewport to be focused (hovered/captured).

## Viewport focus + capture
- Left-click in the viewport captures until release.
- Esc cancels viewport capture and clears selection.

## Camera
Orbit mode:
- Alt + LMB drag: orbit around the focus.
- MMB drag: pan.
- Mouse wheel: zoom.

Free Fly mode:
- RMB drag: look.
- MMB drag: pan.
- Mouse wheel: move forward/back.

Keyboard:
- WASD: move.
- Shift: faster, Ctrl: slower.
- F: focus (orbit mode).

## Navigation
- G: Go To Tile dialog (Enter to submit, Esc to cancel).

## Selection
- LMB click tile/prop to select.
- Click empty space to clear selection.
- Esc clears selection.

## Overlays + snapping
- O: toggle overlays master.
- , / . : cycle snap mode (coarse to fine).
- [ / ] : cycle sub-grid spacing.
- Viewport header -> "Overlay Options" opens a window for per-overlay toggles, including FPS and present mode.

## Debug markers
- "Ray Hit" and "Prop Debug" toggles live in "Overlay Options."
